## image

Generate images from a prompt.

```bash
ai image "a sunset over mountains"
ai image "edit this" < photo.png
```

### Options

<Properties>
  <Property name="-m, --model" type="string">
    Model ID. Comma-separated for multi-model.
  </Property>
  <Property name="-o, --output" type="path">
    Output file path or directory.
  </Property>
  <Property name="-n, --count" type="number" default="1">
    Number of images per model.
  </Property>
  <Property name="-p, --concurrency" type="number" default="4">
    Max parallel generations.
  </Property>
  <Property name="--size" type="WxH">
    Image size (e.g. `1024x1024`).
  </Property>
  <Property name="--aspect-ratio" type="W:H">
    Aspect ratio (e.g. `16:9`).
  </Property>
  <Property name="--quality" type="standard | hd">
    Quality level (OpenAI models only).
  </Property>
  <Property name="--style" type="vivid | natural">
    Style (OpenAI models only).
  </Property>
  <Property name="--no-preview" type="boolean">
    Disable inline image preview.
  </Property>
  <Property name="-q, --quiet" type="boolean">
    Suppress progress output.
  </Property>
  <Property name="--json" type="boolean">
    Output metadata as JSON.
  </Property>
</Properties>

## video

Generate video from a prompt or image.

```bash
ai video "a spinning triangle"
ai image "a dragon" | ai video "animate this"
```

### Options

<Properties>
  <Property name="-m, --model" type="string">
    Model ID. Comma-separated for multi-model.
  </Property>
  <Property name="-o, --output" type="path">
    Output file path or directory.
  </Property>
  <Property name="-n, --count" type="number" default="1">
    Number of videos per model.
  </Property>
  <Property name="-p, --concurrency" type="number" default="2">
    Max parallel generations.
  </Property>
  <Property name="--aspect-ratio" type="W:H">
    Aspect ratio (e.g. `16:9`).
  </Property>
  <Property name="--duration" type="seconds">
    Duration in seconds.
  </Property>
  <Property name="--no-preview" type="boolean">
    Disable inline video frame preview.
  </Property>
  <Property name="-q, --quiet" type="boolean">
    Suppress progress output.
  </Property>
  <Property name="--json" type="boolean">
    Output metadata as JSON.
  </Property>
</Properties>

## text

Generate text from a prompt.

```bash
ai text "explain quantum computing"
ai text "summarize this" < notes.txt
```

### Options

<Properties>
  <Property name="-m, --model" type="string">
    Model ID (`creator/model`). Comma-separated for multi-model.
  </Property>
  <Property name="-o, --output" type="path">
    Output file path or directory.
  </Property>
  <Property name="-f, --format" type="md | txt" default="md">
    Output format.
  </Property>
  <Property name="-n, --count" type="number" default="1">
    Number of generations per model.
  </Property>
  <Property name="-p, --concurrency" type="number" default="4">
    Max parallel generations.
  </Property>
  <Property name="-s, --system" type="string">
    System prompt.
  </Property>
  <Property name="--max-tokens" type="number">
    Maximum tokens to generate.
  </Property>
  <Property name="-t, --temperature" type="number">
    Temperature (0–2).
  </Property>
  <Property name="-q, --quiet" type="boolean">
    Suppress progress output.
  </Property>
  <Property name="--json" type="boolean">
    Output metadata as JSON.
  </Property>
</Properties>

## models

List available models from the AI Gateway.

```bash
ai models
ai models --type image
ai models --creator openai --json
```

### Options

<Properties>
  <Property name="--type" type="text | image | video">
    Filter by modality.
  </Property>
  <Property name="--creator" type="string">
    Filter by creator (e.g. `openai`, `google`).
  </Property>
  <Property name="--json" type="boolean">
    Output as JSON with descriptions.
  </Property>
</Properties>

Models are fetched live from the AI Gateway.

## Timeouts

Requests that exceed the per-command timeout are aborted automatically:

| Command | Timeout |
| --- | --- |
| `text` | 120 seconds |
| `image` | 120 seconds |
| `video` | 300 seconds |

Video generation uses a longer timeout because models typically need more processing time.

## Exit codes

| Code | Meaning |
| --- | --- |
| `0` | Success |
| `1` | All generations failed |
| `2` | Partial failure (some succeeded, some failed) |
